***** MULTIPLE  HUMAN DESIESE PREDICTION ****

* PREREQUISITES TO SET-UP THE WEB APP.
    -> Python should be installed.
    -> Any IDE of python like vscode,  pycharm etc.
    -> Install flask.
    -> Creation of virtual environment is optional.
    -> Install Tensorflow, pandas, os, pip, keras in IDE terminal.

* TO START THE WEbAPP.
    -> Open app.py 
    -> In teminal type "python ./app.py"
    -> Then copy the url like "http://127.0.0.1:5000" and paste it to the url.
